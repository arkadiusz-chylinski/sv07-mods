# sv07-mods
Collection of hardware and firmware modifications applied to my personal Sovol SV07+ 3D printer.

After purchasing my first 3D printer, I decided to progressively upgrade and modernize the machine within a limited budget.
Each modification was implemented to solve a specific issue or improve the printer's performance, reliability or usability.

## Main modifications
### Motion system

- X and Y axis upgrade to linear rails

- redesigned mounting brackets for improved rigidity

### Toolhead

- custom toolhead upgrade

- improved cooling and cable management

- separate toolboard

### Firmware

- firmware reconfiguration using Klipper

### Quality-of-life upgrades

- inductive / eddy current bed probe

- silicone nozzle cleaning brush

- various printed functional parts
