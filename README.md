# sv07-mods
List of mods added to my personal 3D printer

Having bought my first 3D printer on a bargain, I took it upon myself to modernize it the best I could with a budget I could afford. All changes featured here have beed dictated by a particular need or ambition in mind. 

### Main mods:
- X and Y axis linear rails upgrade
- Toolhead upgrade
- Firmware reflash
- QoL upgrades
